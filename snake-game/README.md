# 🐍 Snake Game

Simple Snake Game built with **HTML, CSS, and JavaScript** using the Canvas API.

## 🎮 Gameplay

- Use arrow keys to control the snake
- Eat the red food to grow
- Avoid hitting the wall or yourself

## 🕹 Controls

Arrow Keys

⬆️ Up  
⬇️ Down  
⬅️ Left  
➡️ Right  

## 🚀 Tech Stack

- HTML
- CSS
- JavaScript
- Canvas API

## 📸 Preview

Simple retro snake gameplay.

## 📂 Run locally

Just open:

index.html

in your browser.
